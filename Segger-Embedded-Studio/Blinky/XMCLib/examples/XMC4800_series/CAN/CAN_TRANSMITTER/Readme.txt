CAN Transmitter demo example

Description of example project
==============================
This example shows how to send CAN frames periodically.
The data transmitted is the state of the buttons

Hardware Setup
===============
XMC4700 RelaxKit

How to test the application
============================
a. Import the project 
b. Compile and flash the application onto the device
c. Using a CAN bus monitor you can see the CAN frames being sent periodically 
   or you can connect the second as shown in BOARD_SETUP.png and check the debug information 
   sent over the terminal
