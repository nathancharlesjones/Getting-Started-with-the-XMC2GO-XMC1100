UART SHELL demo example

Description of example project
==============================
This example shows how to implement a shell program using UART as communication channel.

Hardware Setup
===============
XMC1300 BootKit board
Rx Pin : 1.3 , Tx Pin : 1.2 (connected through Debug & Com part to the USB connection)
Use a teminal program and configure the COM port using 115200 baud rate, 8 data bits, 1 stop bit, no parity.
