Description of example project
==============================
The example shows how to trigger externally the transfer of a SPI data word. 

Hardware Setup
===============
XMC4300 EtherCAT Relax Kit
SPI USIC is implemented across the XMC family and therefore the example is easily portable to other boards.

How to test the application
============================
a. Import the project
b. Compile and flash the application onto the device
d. Run the application.
